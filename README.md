## ✨ Features

- 🎬 **Full-screen video hero** on homepage
- 🗺️ **Destination grid** — Udaipur, Jaipur, Jaisalmer, Jodhpur
- 💰 **12 Travel Packages** with pricing, itinerary, inclusions & exclusions
- 🔍 **Filter & Sort** packages by category and price
- 📱 **Fully Responsive** — Mobile, Tablet, Desktop
- ☰ **Full-page split menu overlay** with image hover effect
- ✨ **Scroll reveal animations** throughout
- 📞 **WhatsApp inquiry buttons** on all service pages
- 🌙 **Premium aesthetic** — Cormorant Garamond + DM Sans fonts

---

## 🛠️ Tech Stack

- **HTML5 / CSS3 / Vanilla JavaScript**
- **Font Awesome 6** — Icons
- **Google Fonts** — Cormorant Garamond, DM Sans, Cinzel
- **AOS** — Animate On Scroll library
- No frameworks, no build tools — pure HTML/CSS/JS
